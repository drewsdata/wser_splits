This analysis aggregates available timing data for the Western States Endurance Run (WSER). It is not endorsed by nor affiliated with the WSER. Hopefully, this information can help others achieve their goals who are fortunate enough to run the race.

Resources:
Data source: https://www.wser.org/splits/
Additional analysis: https://www.wser.org/geeks-only/
Relevant gender policies: https://www.wser.org/transgender-entrant-policy/
